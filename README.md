## :hamburger: Como usar

```bash
# Clonar o repositório
$ git clone https://github.com/MatheusINFO/burger-builder.git

# Ir dentro do repositório clonado
$ cd BurgerBuilder

# Instalar as dependências
$ yarn install

# Rodar o app
$ yarn start
```
